NanoQt Documentation

Overview

NanoQt is a program intended for data acquisition and experiment control. It's user interface is based on the JavaScript language (actually Qt Script, which is an implementation of the ECMA-262 specification, just like JavaScript). We use an ADwin system as the main interface to the experimental hardare.
