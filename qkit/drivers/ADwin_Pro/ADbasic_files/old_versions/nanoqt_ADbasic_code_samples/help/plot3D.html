The 3D Plotter

This Qt Widget is a tool to plot surfaces with OpenGL. This very powerful graphic library is multiplatform, and takes benefits of the hardware in your graphic card to plot 3d objects. With the command plot3D you may plot either simple surface of type z=f(x,y), sampled in a matrix, or more complex parametric surface of type (x,y,z)=(f1,f2,f3)(u,v) sampled in three matrices. See the two examples plot3D_f_xy.js and parametric_plot3D.js in the sub-directory /samples

The mouse is interacting with the renderer:

  • Click Left + move : rotates X + Y
  • Click Right + move : rotates X + Z
  • Wheel : zoom in/out
  • Double Click : opens 3D Options window

The png button creates a bitmap in png format wich size is defined in the 3D window options.

Related script commands: plot3D() ; get_options3D() ; get_data() (soon!)
