Adwin setup and Troubleshooting

If Adwin does nothing, check a few simple things:

  • Check that Adwin is plugged onto the proper TCP/IP board, with a cross cable.
  • Check that NanoQt is compiled for the true Adwin and not the virtual one: In the NanoQt.pro file, the line #CONFIG += virtualadwin must start with # so that NanoQt is linked to the true Adwin driver.
  • Check with ADTools if your Adwin system is working or not

Some Adwins PRO do not embbed the same cards: as a consequence the plugged cards are not referenced with the same numbers. To fix the card numbers for both inputs and outputs, use set_io_addresses(ain,aout) instruction. This is useless on an Adwin Gold, there is no card number.

Command description
set_io_addresses(ain,aout) Set the input card number for analog inputs (ain), and the output card number for analog outputs (aout).These numbers should be set once at the beginning of your script.

If you make a program that overload your adwin, you may loose the connexion to it, you have to reboot Adwin, and modify your script in order to reduce the T11 load.
