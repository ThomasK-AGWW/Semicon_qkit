The 3D options

This Qt Widget is a tool to manage plot option surfaces. It is composed of two tabbed-windows. The first one is the Gradient Editor, wich is a tool for defining color gradients , the second one the other options, such as label axis, etc...

Detailed Plot3D options

  • The first ComboBox selects the X,Y or Z axis, to fix in the corresponding textBox its label
  • The second ComboBox selects the ticks size and spacing of X,Y or Z axis, the value is fixed in the corresponding textBox
  • The third ComboBox selects the width and height of the bitmap obtained by pressing the button "png" in the 3D renderer, the corresponding textBox let you choose the number of pixels
  • The last textBox let you fix the number of isoCurves on the surface
  • The two radio buttons toggle between Orthogonal/Conic perspective, no yet implemented

The Apply button reads all the parameters in the window and send the order to replot to the renderer

Related script commands: plot3D() ; get_options3D() ; get_data()
