Functions for data analysis

All the functions below operate on arrays of 2D points, i.e. arrays of the form [[x0, y0], [x1, y1], [x2, y2] ...].

lasso_filter() is meant to select a subset of the data points by manually drawing a contour around the points you want to keep. You would normally use this function together with the pencil tool on a Plot to draw the contour. See samples/lasso+plot.js for a detailed example.

convolve() assumes the abscissa of the data are equally spaced. The convolution kernel is given as an array of numbers, i.e. only the ordinates: [y0, y1, y2, ...]. The abscissa are implicit by assuming the same spacing as in data and the whole kernel being centered at zero. If the length of the kernel is even, then the abscissa of the output will lie in between the abscissa of the original data.

function description
get_maxima(data, min_value, min_separation [, min_position]) Finds the local maxima of data. Keeps only those maxima where y > min_value. Discards a maximum if it is too close and not bigger than a previously found maximum. Returns an array of 2D points with the maxima found.
get_minima(data, max_value, min_separation [, min_position]) Finds the local minima of data. Keeps only those minima where y < max_value. Discards a minimum if it is too close and not smaller than a previously found minimum. Returns an array of 2D points with the minima found.
get_median(data) Finds the x and y median values of data. Returns a 2D point [median(x), median(y)].
get_average(data) Computes the average of the x and y coordinates of the data points. Returns [<x>, <y>].
lasso_filter(lasso, data) Filters the list of data points keeping only those that lie inside the contour defined by lasso. Both data and lasso are arrays of 2D points. The ends of lasso are implicitely connected. Uses non-zero winding rule.
linear_fit(data) Fits a straight line (y = a*x + b) through the data. Returns [a, b].
convolve(data, kernel) Convolves the data with a 1D kernel. Returns the convolution as an array of 2D points.
