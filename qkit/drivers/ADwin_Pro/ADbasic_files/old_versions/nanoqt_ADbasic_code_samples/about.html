NanoQt %VERSION%

Data acquisition and experiment control using Qt.

This program is developed by the Nanospin Group of the Néel Institute. It's main purpose is to control experimental setups for nanophysics work.

We use an ADwin to interface the hardware in real time. NanoQt’ user interface is based on the Qt Script scripting language (Qt’s implementation of JavaScript/ECMAScript). See the help pages for details.

Warning: This executable is compiled to drive a %ADWIN_TYPE% ADwin. You can change this by editing NanoQt.pro and recompiling.
